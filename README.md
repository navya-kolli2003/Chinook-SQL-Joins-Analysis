# Chinook-SQL-Joins-Analysis
“Description” box: SQL Intermediate Task using the Chinook database. This project demonstrates INNER and LEFT JOIN operations to analyze customer orders, product revenue, and category-wise performance, along with business insights and exported results.
Loaded Chinook dataset into MySQL
Verified primary and foreign key relationships
Performed INNER JOIN to combine invoices and customer details
Performed LEFT JOIN to identify customers without purchases
Calculated total revenue per product (track)
Analyzed genre-wise revenue distribution
Answered business questions using JOIN + WHERE + GROUP BY
Exported joined results into CSV
Documented business insights
